# HR-Analytics-Dashboard

📊 HR Analytics Dashboard (Power BI)


📌 Project Overview


This project is an HR Analytics Dashboard created using Microsoft Power BI to analyze employee data and generate meaningful insights.



The dashboard helps in understanding employee behavior, attrition trends, and workforce distribution, enabling HR teams to make better and faster decisions.

🎯 Objectives
To analyze employee data and identify patterns

To monitor employee attrition and retention

To understand workforce demographics

To improve HR decision-making using data visualization

🛠️ Tools & Technologies
Microsoft Power BI

Power Query (Data Cleaning)

DAX (Data Analysis Expressions)

Microsoft Excel

📂 Dataset Information


The dataset includes:

Employee ID

Age & Gender

Department & Job Role

Salary

Attrition Status

Job Satisfaction

📊 Dashboard Features


🔹 Employee Overview
Total Employees

Average Salary

Employee distribution by department and gender



🔹 Attrition Analysis
Attrition rate by department

Attrition by job role

Identification of high-risk areas



🔹 Employee Insights
Job satisfaction analysis

Work-life balance overview

Performance trends

📈 Key Insights
Higher attrition observed in specific departments

Employees with lower job satisfaction are more likely to leave

Certain age groups show higher turnover

Salary and job role impact employee retention

🚀 How to Use
Download the .pbix file

Open it in Power BI Desktop

Use filters and slicers to explore insights

📌 Conclusion


This project shows how HR data can be transformed into interactive dashboards for better understanding and decision-making. It highlights the importance of analytics in improving employee retention and organizational performance.
